## 为什么用 Netunnel

- 更容易开始：下载客户端、登录、创建隧道，几分钟就能拿到公网地址
- 更贴近真实需求：开发联调、回调调试、临时演示、远程访问都能直接承接
- 更适合轻量使用：`0.5 元 / GB`，`1 元起用`，先试再决定是否长期使用
- 更容易管理：设备、隧道和访问地址集中在桌面客户端里处理

## 适合谁

- 需要从外网访问本地开发环境的开发者
- 需要把本地 Web 服务临时暴露给同事或客户的团队
- 需要远程访问家里电脑、NAS、Windows 远程桌面的个人用户

## 3 步上手

1. 下载并打开桌面客户端
2. 登录后添加或选择一台在线设备
3. 创建 `HTTP`、`TCP` 或域名隧道，获取公网访问地址

想先看操作路径，可以直接进入 [快速开始](./quickstart/README.md)。
