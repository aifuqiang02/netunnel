# 快速开始

目标：5 分钟内跑通第一次 `HTTP` 隧道，并拿到一个可以访问的公网地址。

如果你还没有安装客户端，先进入 [下载](../download/README.md) 页面完成安装。

## 开始前准备

1. 一台已经安装好 `Netunnel` 客户端的 Windows 电脑
2. 一个你本地已经跑起来的 Web 服务或 API
3. 一个可以正常登录的账号

## 4 步完成首次体验

### 1. 下载并登录客户端

1. 从 [下载](../download/README.md) 页面下载 Windows 客户端
2. 安装并打开客户端
3. 登录你的账号

### 2. 准备本地服务

先确认你本地服务本身可以访问。

### 3. 创建 HTTP 隧道

1. 在客户端里进入隧道创建入口
2. 选择 `HTTP` 隧道
3. 填写本地服务地址或本地端口
4. 提交创建

### 4. 复制公网地址并验证

创建成功后，复制 `Netunnel` 给出的公网地址并打开一次。

## 成功标准

- 公网地址真的能访问

## 如果没跑通

1. 客户端是否正常登录
2. 本地服务是否真的启动了
3. 是否创建的是 `HTTP` 隧道
4. 公网地址是否真的能打开

更多排查见 [常见问题](../faq/README.md)。

## 建议

- 优先选 `HTTP` 隧道
- 优先映射一个简单的本地网页或 API
