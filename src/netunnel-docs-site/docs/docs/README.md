# 使用文档

这里是 `Netunnel` 的产品使用文档入口。

这一部分不再承担“推广落地”职责，而是承接已经开始使用产品、准备继续深入了解功能的用户。

如果你还没有完成第一次体验，建议先看 [快速开始](../quickstart/README.md)。

## 这部分文档解决什么问题

使用文档主要回答这些问题：

- 设备怎么管理
- 不同隧道类型怎么选
- 隧道怎么配置
- 桌面客户端里各项设置是什么意思
- 用量和计费去哪里看

## 推荐阅读顺序

如果你是第一次深入看文档，建议按这个顺序：

1. [快速开始](../quickstart/README.md)
2. 设备管理
3. `HTTP` 隧道
4. `TCP` 隧道
5. 客户端设置
6. 用量与计费

也就是说，先把“能不能跑通”解决，再看更细的配置说明。
