## 安装说明

1. 下载 `Netunnel` 桌面客户端安装包
2. 双击安装并完成首次启动
3. 登录客户端
4. 添加或选择一台在线设备
5. 创建一个 `HTTP`、`TCP` 或域名隧道开始使用

如果你只是第一次体验，建议先从 [快速开始](../quickstart/README.md) 进入，先跑通一次最简单的使用流程。

## 安装前建议

- 优先在一台自己可控的 Windows 设备上首次体验
- 准备一个可以被映射的本地服务，例如本地 Web 页面或 API 服务
- 如果只是为了先体验一次，优先创建 `HTTP` 隧道
- 首次使用时，建议同时打开 [快速开始](../quickstart/README.md) 页面
